\subsection{Detailed Description}
DTSTAT -- Print a status summary of the DTS network.

Usage: dtstat \mbox{[}options\mbox{]}

Options: -h print help

-c $<$cfg$>$ specify the config file to use -p $<$port$>$ specify server port

-e \mbox{[}$<$N$>$\mbox{]} number of errors to print -q $<$name$>$ queue name -s print one-line summary only -t target machine

-v verbose
