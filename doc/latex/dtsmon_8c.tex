\subsection{Detailed Description}
DTSMON -- Simple RPC client to log messages from DTS servers.

Usage:

dtsmon \mbox{[}-i\mbox{]} \mbox{[}-p $<$N$>$\mbox{]} \mbox{[}-version\mbox{]} \mbox{[}$<$log\_\-file$>$\mbox{]}

-i run interactively -p monitor port number

-version print build version string

\mbox{[}log\_\-file\mbox{]} message log file
