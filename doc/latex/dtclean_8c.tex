\subsection{Detailed Description}
DTCLEAN -- Clean the source files or DTS working directory.

Usage: dtclean \mbox{[}options\mbox{]}

Options: -h print help

-c $<$cfg$>$ specify the config file to use -p $<$port$>$ specify server port

-t trace -v verbose
